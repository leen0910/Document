接口说明

  • 1.接口数据全部采用JSON格式
  • 2.JSON字段都采用小写字母,数字命名，单词间采用“_”连接,JSON字符串值中的单位采用标准大小写形式。
  • 3.默认JSON字段都必须填写，加？号的字段为选填字段。
  • 4.所有接口返回分下面三种情况：
    • 4.1 如果有数据返回对应的JSON数据包。
    • 4.2 所有接口正确无数据时返回 示例： js { "action": "device.control", "data": { "success": true } }
      • success: bool 一直为true，如果有错误时应该按照错误格式返回
    • 4.3 所有接口错误时返回示例： js { "action": "device.info.read", "error": { "status": 1005, "description": "read file error." } }
      • status: uint 错误码
      • description: string 错误描述信息u(英文简单描述)

1.1 读取设备信息 命令：device.info.read

说明：搜索设备时发送的命令

UI发送数据 示例：

{
  "action": "device.info.read",
  "data": {
    "type": "server",
    "time": "2017-01-07 08:02:56"
  }
}
  • type: string 连接的设备类型（server：转发服务器，pc:电脑，pad：安卓平板，android:安卓手机，ios:苹果手机）默认为pc电脑连接。
  • time: string UI时间，用于控制器开机后设置一次系统时间（格式：2017-01-07 08:02:56 ）

主控正确返回 示例：

{
  "action": "device.info.read",
  "data": {
    "name": "dev1",
    "group": "group1",
    "serial": "51406870-fc49-4bba-ab96-77ba9f1ebb9d",
    "controlled": false,
    "username": "管理员",
    "ui_ip": "192.168.1.129",
    "status": 0,
    "version": "1.21.1",
    "program": "111",
    "ip": "192.168.1.29",
    "model": "QX Robot 4 Axis"
  }
}
  • name: string 设备名称。
  • group: string 分组名称
  • serial: string 设备序列号
  • controlled: bool 已经被控制时为true，可控时为false
  • username: string 控制者名字（没有被控制时为空）
  • ui_ip: string 控制者ip
  • status: int 设备状态（0:未初始化,1：初始化完成 2：执行程序 3：故障）
  • version: string 主控版本号
  • program_name: string 设备正在运行的程序名称（没有运行时为空）
  • ip: string 设备ip
  • model: string 机器人型号

1.2 修改设备信息 命令：device.info.change

UI发送数据

{

  • password: string 新控制密码
  • ip: string 设备IP

      "current": 1.235,
      "velocity": 32.25,
      "position": 1.235,
      "acceleration": {
        "x": 0.1,
        "y": 0.1,
        "z": 0.1
      }
    },
    "joint2": {
      "voltage": 48.022,
      "current": 1.235,
      "velocity": 32.25,
      "position": 1.235,
      "acceleration": {
        "x": 0.1,
        "y": 0.1,
        "z": 0.1
      }
    },
    "joint3": {
      "voltage": 48.022,
      "current": 1.235,
      "velocity": 32.25,
      "position": 1.235,
      "acceleration": {
        "x": 0.1,
        "y": 0.1,
        "z": 0.1
      }
    },
    "joint4": {
      "voltage": 48.022,
      "current": 1.235,
      "velocity": 32.25,
      "position": 1.235,
      "acceleration": {
        "x": 0.1,
        "y": 0.1,
        "z": 0.1
      }
    }
  }
}
  • joint1: 关节1信息
    • voltage: float 每个轴的当前电压（单位：V）
    • current: float 每个轴的当前电流（单位：A）
    • velocity: float 每个轴的当前电压（单位：m/s）
    • position: float 每个轴的当前位置值（单位：x/y/r为rad，z轴为mm）
    • temperature: float 温度
    • acceleration: 加速度
      • x: float X方向加速度
      • y: float Y方向加速度
      • z: float Z方向加速度
  • joint2: 关节2信息，字段同关节1
  • joint3: 关节3信息，字段同关节1
  • joint4: 关节4信息，字段同关节1

3.2 获取初始化状态 命令：device.initialize.check

每1秒查询

UI发送数据

{
  "action": "device.initialize.check",
  "data": {}
}

主控正确返回 示例：

{
  "action": "device.initialize.check",
  "data": {
    "mode": 0,
    "stage": 0,
    "connect": {
      "joint1": true,
      "joint2": true,
      "joint3": true,
      "joint4": true,
      "main_io": true,
      "tool_io": true
    },
    "init": {
      "joint1": true,
      "joint2": true,
      "joint3": true,
      "joint4": true
    },
    "excuting": {
      "joint1": true,
      "joint2": true,
      "joint3": true,
      "joint4": true
    },
    "result": {
      "joint1": true,
      "joint2": true,
      "joint3": true,
      "joint4": true
    }
  }
}
  • mode: uint寻找机械位置模式（0：自动，1：手动，2：拖拽）
  • stage: uint 初始化步骤（0：未执行 1：正在执行 2：执行完成 3：执行错误）
  • connect: 连接状态
    • joint1: bool 关节1是否连接
    • joint2: bool 关节2是否连接
    • joint3: bool 关节3是否连接
    • joint4: bool 关节4是否连接
    • main_io: bool 主IO是否连接
    • tool_io: bool 工具IO是否连接
  • init: joint1/joint2/joint3/joint4:bool 关节是否初始化
  • excuting: joint1/joint2/joint3/joint4: bool 正在执行初始化的关节
  • result: joint1/joint2/joint3/joint4: bool 关节是否找到机械位置

6.1 设置IO输出 命令：device.io.write

UI发送数据

{
  "action": "device.io.write",
  "data": {
    "version": "FFA50001",
    "d_out": [
      {
        "channel": 1,
        "value": 0
      },
      {
        "channel": 2,
        "value": 0
      }
    ],
    "da": [
      {
        "channel": 1,
        "mode": "current",
        "value": 2
      }
    ],
    "ad": [
      {
        "channel": 1,
        "mode": "5v"
      },
      {
        "channel": 2,
        "mode": "current"
      }
    ],
    "dof": [
      {
        "channel": 1,
        "mode": "digital",
        "value": 0
      }
    ]
  }
}
  • version: string IO版本名称（"FFA50001":主IO板；"FFA60001":工具IO板；）
  • d_out: 数字输出
    • channel: uint 通道
    • value: uint 设置值（0: 关闭；1: 打开）
  • ad: 模拟输入
    • channel: uint 通道
    • mode: string 模式（"current":电流 / "5V":电压5V / "10V":电压10V）
  • da: 模拟输出
    • channel: uint 通道
    • mode: uint 模式（"close":关闭 / "current":电流 / "voltage":电压）
    • value: float 输出值（电压单位：V / 电流单位：mA）
  • dov: 数字电压输出
    • channel: uint 通道
    • mode: string 模式（"close":关闭 / "12V":电流 / "24V":电压）
  • dof: 功能数字输出
    • channel: uint 通道
    • mode: string 模式（"digital":数字输出 / "pwm":电流 / "frequency":频率）
    • value: uint 设置值（0: 关闭；1: 打开）
    • frequency: float 频率值（用于frequency/pwm模式）
    • count: uint 频率输出个数（用于frequency模式，取值范围）
    • cycle: float PWM占空比（用于pwm模式,取值范围0-100表示百分之0-100）

主控按照正确无数据或错误返回

10.1 附录错误信息对照表

错误码用5位整型值表示：

10000 - 20000 普通信息 20000 - 30000 警告信息 30000 - 40000 错误信息 40000 - 50000 严重错误

  • 10000 上电失败
  • 10001 设备急停
  • 10002 握手超时
  • 10003 驱动器ping-pong超时
  • 10004 驱动器自动找位置超时
  • 10005 驱动器机械码盘错误
  • 10006 加速度过大急停
  • 10007 驱动器自检初始化超时

警告码 描述

  • 50000 驱动器缓存清空超时
  • 50001 驱动器未上电命令无效
  • 50002 控制器急停命令无效
  • 50003 配置未加载
  • 50004 正在执行拖拽命令无效
  • 50005 未找到机械位置，命令无效
  • 50006 零点未加载命令无效
  • 50007 指令位置，或运动过程中超限
  • 50008 驱动器未到达目标位置
  • 50009 程序内容错误
